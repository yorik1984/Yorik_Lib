Описание

Плагин для Sketchup® 2014 Pro или выше. Представляет собой библиотеку для использования другими плагинами. Также включает в себя YorikLangHandler - доработанный инструмент, в первую очередь для собственных нужд, из стандартного набора Sketchup® API LanguageHandler для локализации плагинов. Для всех плагинов и инструментов автора(Yorik Tools) будет использован язык(локализация), которая будет выбрана в соответствующем меню:

Tools ► Yorik Lib ► Язык интерфейса

Более подробно из принципом работы можно на странице Yorik LangHandler.

Описания работы всех методов для разработчиков (на англ. языке) на rubydoc.info.

Использование

Использовать можно как отдельный плагин путем подключения только модулей так и отдельные файлы или часть исходного кода для разработки собственных плагинов. Меню доступа к инструментам находится по пути:

Tools ► Yorik Lib

Установка

Плагин работает только в Sketchup® 2014 Pro или выше. Установите новую версию программы перед использованием плагина. Installing Ruby Plugins in Sketchup.
