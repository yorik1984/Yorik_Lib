Описание

Плагин для Sketchup® 2014 Pro или выше. Доработанный инструмент, в первую очередь для собственных нужд, из стандартного набора Sketchup API LanguageHandler для локализации плагинов.

Общее из стандартным инструментом:

  • принцип создание локализованных строк аналогичен оригинальному инструменту
  • возможность использовать одни и те же файлы локализации .strings

Некоторые отличия от стандартного:

  • смена языка(локализации) интерфейса(для вступления в силу нового перевода необходим перезапуск)
  • язык интерфейса плагина не зависит от текущего языка программы

Описания работы всех методов для разработчиков (на англ. языке) на rubydoc.info.

Использование

Использовать можно как отдельный плагин путем подключения только модулей так и отдельные файлы или часть кода для разработки собственных плагинов.

Установка

Плагин работает только в Sketchup® 2014 Pro или выше. Установите новую версию программы перед использованием плагина. Installing Ruby Plugins in SketchUp.
