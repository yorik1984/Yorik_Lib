About

Plugin for Sketchup Pro 2014 or higher. It is a library for use by other plugins. Also includes YorikLanguageHandler - modified tool, primarily for personal using, of a standard Sketchup API LanguageHandler for plugins localization.

Code documentation can be finding on the rubydoc.info.

Using

Can be using as a plugin or by connecting the modules and individual files or a part of code to develop own plugins.

Installing

Plugin works only in Sketchup 2014. Update SketchUp before install. Installing Ruby Plugins in SketchUp.
