Plugin for SketchUp 2014 Pro and above "Add attribute"

Description

Plugin for Sketchup 2014 Pro or above is to define dynamic attributes of selected components of the model, including components in all levels of nested components.

Features :

  • Adding standard attributes
  • Adding custom attributes
  • Works only with the components
  • Dynamically change levels of nested components depending on the selection

Usage

You can use the plugin from toolbar

or from the main menu.

Extensions > Add attribute

  1. Add attributes inputbox - It includes a dialog box with a drop-down list to select the attributes you want to add.
  2. Help - Open the help file (this page).

When selecting not only components you will see a warning message appears that you need to "Select only components".

Note. For a speed selection of components, there are several different plugins. For example, Selection Toys.

If the choice is made correctly, you will see a dialog box where you can select an attribute to set its value. It may be custom (Custom...), choose from a list of standard and the possibility of switching units in components by setting attributes (Toggle Units).

When creating a custom attribute (Custom...), there is an additional window to enter the name.

The attribute name has a number of limitations. If it is an incorrect name, a format will be warning with an explanation of the error as you type. For example, an attribute name with a beginning number.

Below is a list of all errors entering the name:

  • Attribute name can't be empty

  • Attribute name can't contain spaces

  • Attribute name can only contain Latin letters and numbers

  • Attribute name can't begin with an underscore

  • Attribute name can't begin with an number

  • You may not name an attribute TRUE or FALSE

If the entered name matches with the standard attributes, input attribute values will be as the standard with the same name. For standard attributes dialog box will contain other input fields, depending on the attributes.

The input fields are filled as well as a built-in editor attributes. Some fields are added for convenience and do not have analogues. Some can have a help in brackets for understanding of how they work.

  • List Option. Entering values is carried out on a template &Opt1=Val1&Opt2=Val2&. Recent badges ampersand ("&") required. Between the blocks which are similar to the input line in the standard interface also has to be a separated ampresand. If the value will mean length, the need to set in inches.

  • Duplicate attribute name. A method of processing attribute, if already presented:

    - Ignore - The attribute value will remain the same

    - Replace - The attribute value will be replaced by a new one

    - Equal replace - It will be processed only the attribute value if the attribute exists. If there is, then it will not be assigned. If there is, it will set a new value.

  • Component nesting levels. The nesting level of processed components. If you select the maximum number from the list, it will be processed all values. For more information on nesting, you can see in the figure below.

  • Report:

    - Off - Deny report

    - Short in messagebox - A brief report in the information window

    - Full in console - Full report in the Ruby Console (Window > Ruby Console). The report will be displayed: the name of the attribute, the number of components, which has been applied an operation job attributes, the nesting level components. Group can add a level of geometry modeling, but the group can not be added to the attribute. The list of components is displayed in a tree according to the nesting elements, similar in Outliner.

Installation

Plugin works on Sketchup 2014 Pro and above. Update SketchUp Version before installing the plugin. Installing Ruby Plugins in SketchUp
