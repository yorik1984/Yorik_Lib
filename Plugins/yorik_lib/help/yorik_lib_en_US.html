About

Plugin for Sketchup® Pro 2014 or higher. It is a library for use by other plugins. Also includes YorikLanguageHandler - modified tool, primarily for personal using, of a standard Sketchup® API LanguageHandler for plugins localization. All author's tools and plugins (Yorik Tools) will be using locale which will be selected in:

Tools ► Yorik Lib ► Language translation

More information about using localization tool can be finding in Yorik LangHandler.

Documentation of source code located on the rubydoc.info.

Using

Can be using as a plugin or by connecting the modules and individual files or a part of source code to develop own plugins. Menu to access for tools is here:

Tools ► Yorik Lib

Installing

Plugin works only in Sketchup® 2014. Update before install. Installing Ruby Plugins in Sketchup.
